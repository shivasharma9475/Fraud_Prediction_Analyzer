About Fraud prediction analyzer

What is Fraud Prediction Analyzer?

Fraud Prediction Analyzer is an interactive web application that allows users to input transaction details and instantly check whether a transaction is likely to be fraudulent or not. It also provides a probability score indicating the likelihood of fraud.

How It Works

Users enter key transaction information such as amount, originator balance ratio, transaction type, time of day, and certain binary flags. The application sends this data to a backend model, which evaluates the inputs and returns a prediction and probability score in real time.

Features

  • Interactive form for simplified transaction input.
  • Real-time prediction: Fraud or Not Fraud.
  • Probability score for risk assessment.
  • Clear, color-coded results for high/low risk transactions.
  • Mobile-friendly, responsive design.

Purpose

This app helps financial institutions or users quickly identify suspicious transactions, enabling proactive fraud detection and decision-making.

Technology Stack

Frontend: HTML, Tailwind CSS, JavaScript
Backend: Python (Flask)
Model: Pre-trained machine learning model for fraud prediction
